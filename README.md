# Inclusive Citizen AI Assistant

VHACK 2026 Project

## Problem
Many citizens cannot understand complex government documents due to language barriers and complicated terminology.

## Solution
We built an AI assistant that:
- Retrieves information from official documents
- Simplifies complex language
- Translates answers into multiple languages

## Technologies Used
- Python
- Streamlit
- LangChain
- FAISS
- HuggingFace Models

## Features
- Question answering from government documents
- Simplified explanations
- Multilingual translation
- Web interface

## How to Run

1. Install requirements
pip install -r requirements.txt

2. Run the app
streamlit run main.py
